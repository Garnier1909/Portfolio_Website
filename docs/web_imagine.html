イマジン 公式ホームページ

概要 大妻女子大学のオープンキャンパスで公開した、マルチスクリーンインスタレーション
『イマジン』のホームページです。
デザイン オープンキャンパスに使用するサイトという事で、キーカラーは大学のものである紫に寄せました。 また、社会情報学部の各専攻を紹介する役割があるため、それぞれの専攻にいる人物をイメージカラーで分けて特徴を際立たせました。

大学内に掲示されているQRコードからアクセスできる仕様となっているため、スマートフォンでの閲覧を想定しています。
担当 デザイン / コーディング / 一部ライティング
制作期間 1週間程度
使用ツール Xd / Visual Studio Code
